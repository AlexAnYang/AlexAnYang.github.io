    # FORMAT,ROUND和TRUNCATE的区别

    # 1、ROUND(X,D) 可以进行四舍五入，按照对应的位数

    此函数返回x舍入到最接近的整数。如果第二个参数，D有提供，则函数返回x四舍五入至第D位小数点。 D必须是正数或全部右边小数点的数字将被删除

    # 2、TRUNCATE(X,D) 直接截取，不进行四舍五入

    此函数用于返回X的截断到小数位D号的值。 如果D为0，则小数点被除去。如果D是负的，那么D的值的整数部分值的数量被截断

    # 3、FORMAT(X,D) 会四舍五入，返回类型是字符串，因为满3位会加一个逗号

    FORMAT(X,D) 函数用于以下格式格式化数字X：###，###，###。##截断为D位小数

    举例如下

    SELECT  FORMAT(2687.6487,2), ROUND(2687.6487,2),TRUNCATE(2687.6487,2)
