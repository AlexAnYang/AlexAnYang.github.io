# Golang的安装配置golint

# 1 下载并编译golint

# 方法一 官网下载代码编译成golint.exe

  • 下载golang 的 lint，下载地址：https://github.com/golang/lint
  • 解压文件到$GOPATH/src/github.com/golang/lint
  • 到目录$GOPATH/src/github.com/golang/lint/golint中运行go build ./
  • 在当前目录有golint的可执行程序
# 当然，最简单的方式是：
go get github.com/golang/lint
go install github.com/golang/lint
